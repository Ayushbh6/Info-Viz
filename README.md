# Info-Viz

Information Visualization group project for TU Wien.

## Project Scope

This repository currently contains the proposal-stage materials for a UFC-focused information visualization project:

- `UFC_InfoViz_Proposal.md`: proposal draft
- `UFC_InfoViz_Proposal.pdf`: exported proposal
- `ufc_master_scraper.py`: scraper for UFC statistics data
- `Project.txt`: course project notes and deadlines
